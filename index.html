      Addressing Issues with Working Memory in Video Object Segmentation

      This study introduces a simple algorithmic adjustment to enhance video object segmentation (VOS) models by improving their handling of inconsistent camera views, such as sudden cuts or extreme context shifts, which typically disrupt working memory in VOS. By selectively updating memory only when the target object is in frame, this modification significantly enhances model performance on challenging video data, thereby broadening the practical applicability of VOS models in real-world scenarios.

      Passive Sonar Source Classification and Localization

      This paper proposes a model pipeline to enhance passive sonar navigation for submarines and naval vessels, addressing challenges in contact classification, localization, and distance approximation under noisy aquatic conditions. By combining convolutional neural networks (CNN) for signal discrimination, 2D image filtering for bearing determination, and a novel LSTM model for distance estimation based on bearing rate alone, this approach aims to advance towards fully autonomous sonar-based navigation.

      Deepfake Modeling

      This paper explores different models for generating Deepfake images while also exploring how facial features can be manipulated to create convincing fake images. Through implementing two different models for facial generation as well as two different forms of facial manipulation, the paper compares and contrasts the different techniques used to generate fake faces through the use of GANs, diffusion models, and encoder-decoder architectures.
